📋 项目与人员匹配系统
智能任务分配 · 能力管理 · 人员调度

🚀 快速安装
环境要求： Node.js ≥ 16.0 + npm ≥ 7.0

# 1. 克隆项目
git clone https://github.com/你的用户名/仓库名.git
# 2. 进入目录
cd project-assignment-system
# 3. 安装依赖
npm install
# 4. 启动项目
npm run dev

💡 提示： 启动后访问 http://localhost:5173 即可使用。
项目使用浏览器本地存储，无需配置数据库，开箱即用。
✨ 核心功能
🎯 能力管理
管理布尔能力（如技术栈）和数值能力（1-5分评分），支持增删改查

👥 人员管理
管理员工信息、语言技能、能力分数，构建人才库

📌 任务管理
创建任务、定义能力要求、设置负责人、分配开发人员

🤖 智能匹配
根据任务要求自动推荐合适的负责人和开发人员

⚙️ 系统设置
配置核心技能、允许负责人名单、自定义提示信息

💾 本地存储
数据自动保存在浏览器，关闭页面不丢失，无需后端

🛠️ 技术栈
Vue 3 Vite Pinia Vue Router Element Plus LocalStorage
📦 常用命令
开发模式
npm run dev
生产构建
npm run build
预览构建结果
npm run preview
代码检查
npx eslint .
🔐 权限说明
👤 访客 (guest)
仅可查看数据，不可编辑

✏️ 编辑者 (editor)
可增删改查能力、人员、任务，可分配人员

👑 管理员 (admin)
全部权限 + 系统设置、负责人白名单配置

📖 使用流程
进入系统后，先在【能力管理】添加布尔能力（如Vue、React）和数值能力（如前端技能）
在【人员管理】添加员工，配置他们的技能和评分
在【任务管理】创建任务，勾选所需能力和最低分数要求
系统自动推荐负责人和开发人员，确认后完成分配
管理员可在【系统设置】配置核心技能、允许负责人等

💡 温馨提示：
• 所有数据保存在浏览器本地，清除浏览器缓存会导致数据丢失，建议定期导出备份
• 首次使用会自动加载示例数据，方便体验完整功能
• 如需重置数据，在浏览器控制台执行 localStorage.removeItem('projectSystem') 并刷新页面
